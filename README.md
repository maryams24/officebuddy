# Office Helpdesk Buddy

A simple Streamlit helpdesk chatbot that suggests quick fixes and lets users create and track support tickets.
Features includes:
Theme switcher (e.g., Playful Pastel / Neon Pop)
One-click issue shortcuts: Login, VPN, Wi‑Fi, Software, Phishing
Ticket creation form with an issue summary field
Urgency selection (low / med / high)
Optional user context capture (e.g., name, device, OS) to enrich tickets
Clean, form-based submission (Ctrl+Enter hint for faster submit)
